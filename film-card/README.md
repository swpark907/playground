# 감성 필름 명함

아날로그 필름 형식으로 표현된 디지털 명함입니다. 필름의 각 프레임은 프로필 정보를 담고 있으며, 카메라 필름을 넘기는 듯한 감성적인 UX를 제공합니다.

## 기능

- 카메라 뷰파인더 형태의 시작 화면
- "Play My Film" 버튼으로 필름 명함 시작
- 좌우 화살표로 필름 프레임 전환
- 각 프레임에 프로필 정보 표시
- 배경에 흑백 이미지 또는 영상 재생
- 배경 음악 재생

## 사용 방법

1. 프로젝트를 웹 서버에 업로드하거나 로컬에서 실행
2. 필요한 이미지와 영상을 `assets/images/`와 `assets/videos/` 폴더에 추가
3. 배경 음악을 `assets/background-music.mp3`에 추가
4. `script.js` 파일의 `filmData` 배열에서 프로필 정보 수정

## 파일 구조

```
film-card/
├── index.html       # HTML 구조
├── style.css        # 스타일 정의
├── script.js        # 필름 카드 동작 로직
└── assets/
    ├── images/      # 배경 이미지 파일들
    ├── videos/      # 배경 영상 파일들
    └── background-music.mp3  # 배경 음악
```

## 참고 사항

- 필름 프레임 데이터는 `script.js` 파일의 `filmData` 배열에서 관리됩니다.
- 각 프레임에는 제목, 내용, 배경 이미지 또는 영상이 포함될 수 있습니다.
- 키보드 좌우 화살표 키로도 프레임을 전환할 수 있습니다.
